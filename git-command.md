1) To create git repo
 
Command : git init

2) To check the current status of git 

Command : git status

3) To add changes for tracking history 

Commands: git add <file_name>

4) after adding it into staging, to commit for tracking history 

Command: git commit -m "Initial commit"

## branching commands

1) check current branch(default)

Command : git branch (master)

2) Create a new branch as well switch 

Command : git checkout -b <branch_name>

3) switch one to other 

command : git switch <branch_name>
